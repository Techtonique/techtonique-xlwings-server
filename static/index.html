Simple xlwings server for Techtonique API

Get your Techtonique API token first

To get your Techtonique API token and be able to do Machine Learning directly from your desktop, you need to register at https:www.techtonique.net, login, and navigate to https:www.techtonique.net/token. Thanks to the way it was designed Techtonique API allows you to do Machine Learning with any language that you like.

In VBA Excel, create

See example xlsm file 

                    
                        Sub SampleRemoteCall()

                        Dim BASE_URL As String
                        Dim token, token_query As String
                        Dim file_name, file_name_query As String
                        Dim api_route, api_route_query As String
                        Dim other_query, full_query As String
                        
                        ' provide only these 3
                        BASE_URL = "https://techtonique-xlwings-server-1793f0ffd160.herokuapp.com/"
                        token = "Y0UR_TOKEN"
                        file_name = "https://raw.githubusercontent.com/Techtonique/datasets/refs/heads/main/time_series/univariate/AirPassengers.csv"
                        api_route = "forecastingreglinear"
                        
                        token_query = "token=" & token
                        file_name_query = "&filename=" & file_name
                        api_route_query = "&apiroute=" & api_route
                        other_query = "&method=LarsCV"
                        full_query = BASE_URL & "forecastingapicall?" & token_query & file_name_query & api_route_query & other_query
                        
                        RunRemotePython full_query, apiKey:="DEVELOPMENT"
                    End Sub
